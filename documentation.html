CSSframework Documentation

Here you can find documentation for this framework

First step

Before you start do design your application, you need to do this in order to use our framework:

<link rel="stylesheet" type="text/css" href="cssframework.css" />

After you link to cssframework.css file you need to define a .container class to a <div> element. After you do this, you can style your page. Good luck!

Headers

With our framework you can define page headers with ease like this:

This is a page header

This is a subheader

To a <header> element you need to set .header class. After that, in the brackets of <header> element you need to put a <h1> and a <h3>(optional) heading.

Also with our framework you can define article headers like this:

This is an article header

You can define this type of header by giving the .article-header class to a <div> element. Inside this element you need to write the <h2> heading element.

Status paragraphs

This framework allow you to create paragraphs that show the status of an action you do.

This is a status-primary paragraph

This is a status-success paragraph

This is a status-info paragraph

This is a status-warning paragraph

This is a status-danger paragraph

In order to use these status paragraphs you need to create a <p> element and give it a class that contain the status you want to show.

Primary: .status-primary
Success: .status-success
Info: .status-info
Warning: .status-warning
Danger: .status-danger

Input

To style input box you can use .input-box class on an input box.

To style textareas simply use .input-content class.

Buttons

Anchor

For buttons you can use .btn .btn-default classes. To change the size of a button you can use .btn-large and .btn-small, For a medium size you use .btn-default.


To syle on different colors you need to use class .btn-primary, .btn-success, .btn-info, .btn-warning, .btn-danger.

Panel

This is panel title

This is panel body

This is a panel footer

To use panels you have to define a <div> element with .panel class. Inside this element insert a <header> element with a .title class. After that insert a <div> element with a .body class and inside this body you can use a <p> element for content. Optionally, you can use a <footer> element with a .footer class.

Badge

Inbox 5

You can simply use a badge by giving the .badge class to a <span> element.

Labels

Default Primary Success Info Warning Danger

To create a label simply give the .label .label-default class to a <span> element for a default style. Also you can use .label .label-primary, .label .label-success, .label .label-info, .label .label-warning, .label .label-danger to show status of your actions.

Thumbnails

To use thumbnails you can simply add .thumbnail-medium, .thumbnail-larger or .thumbnail-small to a <div> element. Inside it, just simply use a <img> element.
