CSSframework

Simple CSS framework to help users style their page

Download this project as a .zip file Download this project as a tar.gz file

First step

You need to link your page to cssframework.css by inserting this line to your code: <link rel="stylesheet" type="text/css" media="screen" href="stylesheets/cssframework.css">

After you link your page to cssframework.css you need to insert in <body> tag a <div> container with .container class like this:

<body> <div class="container"> ....... </div> </body>

After you finish this step you can start design your pages. Good luck!

Page header

Use .header on a <header> container and then inside this container write <h1> and <h3> tags for header and subheader
